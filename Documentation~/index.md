# Jobs Profiler package

Collect data about jobs in your application to inform areas of optimization.

The Jobs Profiler provides an updated view of the [CPU Profiler](xref:um-profiler-cpu), with detailed information about how [jobs](xref:um-job-system) affect the performance of your application.
